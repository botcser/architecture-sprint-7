Взаимодействие между системами предприятия SmartVideoDoor, SmartBigStick и внешней платформой партнёров:
Платформа (шлагбаум и дверь) предоставляет API по взаимодействию (открытие, ping, состояние...) по HTTPS, и мы аналогично API к SmartVideoDoor, SmartBigStick;
При процессе входа в дверь/проезда под шлагбаумом открывается SFTP соединение, в рамках которого происходит полный цикл, со всеми таймаутами, мониторингом и уведомлениями.
